# /public — Static assets

Place the following files here:

## Required
- **resume.pdf**  
  Your downloadable CV. Linked from the Hero "Resume ↓" button and the Nav.

## Recommended
- **avatar.jpg** (or .png)  
  Your profile photo. Use in the About section once you add an `<Image>` tag.
  Ideal size: 400×400px, square crop.

- **og-image.png**  
  Open Graph image for link previews (WhatsApp, Slack, Twitter, LinkedIn).  
  Standard size: 1200×630px.  
  Once added, set it in `app/layout.tsx`:
  ```ts
  openGraph: {
    images: [{ url: "/og-image.png", width: 1200, height: 630 }],
  },
  ```

- **favicon.ico**  
  Browser tab icon. Next.js picks this up automatically from `/public/favicon.ico`.  
  Or place an `icon.png` in `/app/` for the App Router favicon convention.

## Optional
- **robots.txt** (auto-generated by Next.js if you export it from app/robots.ts)
- **sitemap.xml** (auto-generated from app/sitemap.ts)
